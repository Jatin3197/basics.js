Types in Opertors in JavaScript

  • Arthematic Opertor-

    +(Addition),-(subtraction),*(multiplication),**(exponential),/(division),%(Remainder),++(Increment),--(decrement)
    For Ex-const number = 3 + 5; // 8

  • Assignment Opertors-

    =,+=,-=,*=,/=,%=,**=.
    For Ex-const x = 5;

  • Comparison Opertor-

    ==(equal to),!=(Not equal) many more.
    For Ex-const a = 3, b = 2; console.log(a > b); // true

  • Logical opertor-

    &&(and),||(or),!(not)
    For Ex-const x = 5, y = 3; (x < 6) && (y < 5); // true

  • Bitwise Opertor

    &(Bitwise And),|(Bitwise OR) ,^(Bitwise XOR) many more.
    For Ex-const a = 5; // 00000000000000000000000000000101 const b = 3; // 00000000000000000000000000000011 console.log(a & b); // 00000000000000000000000000000001 // Expected output: 1
