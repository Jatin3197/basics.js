Differnce between HTML and JavaScript

HTML

HTML is a markup language of the webpage.It only make the structure or skelton of the website.It can define the content of the webpage.In the HTML uses many tags to create the structure of the website.It cannot perform the logic in the webpage ,it only make a structre to the webpage.HTML runs on client side.HTML is not case sensitive.

JavaScript

JavaScript is a programming language to perform some tasks for write the code and used for creating dynamic and creative webpages.JavaScript is used to add interactivity, modify the content and styling of a web page, and handle user events. Javascript can perform the logic in the webpages to perform some tasks.JavaScript is a scripting language and can perform logic, make decisions, and manipulate the HTML DOM.Javascript is case sensitive. JavaScript can runs on both the client and server side.
