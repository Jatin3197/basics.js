Primitive Data-types

  1. String--

    A String is series of characters like "harry yert".Strings are written with quotes. You can use single or double quotes.

  2. Number--

    All JavaScript numbers are stored in decimal numbers.Numbers can be written with, or without decimals

  3. Boolean--

    In JavaScript boolean are used just two values true or false.Boolean are used in the condition type statement for true or false values.

  4. Null--

    In null data types are stored empty values.

  5. Undefined--

    In JavaScript, a variable without a value, has the value undefined.
